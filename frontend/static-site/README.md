Found-It static site
====================

This folder contains a pure HTML + CSS static front-end for the Found-It project.

Pages:
- `home.html`, `login.html`, `signup.html`, `forgot-password.html`, `upload.html`, `item-details.html`, `dashboard.html`, `about.html`, `contact.html`

Single stylesheet: `assets/css/styles.css`

To preview live with reload: run `npx live-server --port=5175` (or open `home.html` directly).
